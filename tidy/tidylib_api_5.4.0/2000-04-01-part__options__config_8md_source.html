1 # Tidy Options and Configuration {#tidy_config}
2 
3 ## From the Terminal (Console)
4 
5 To get a list of available options, use:
6 
7 `tidy -help`
8 
9 To get a list of all configuration settings, use:
10 
11 `tidy -help-config`
12 
13 To read the help output a page at time, pipe it to a pager, e.g.,:
14 
15 ~~~
16 tidy -help | less
17 tidy -help-config | less
18 ~~~
19 
20 Single-letter options other than `-f` may be combined; for example:
21 
22 `tidy -f errs.txt -imu foo.html`
23 
24 
25 ## Using a configuration file
26 
27 The most convenient way to configure **Tidy** is by using separate configuration
28 file.
29 
30 Assuming you have created a **Tidy** configuration file named **config.txt**
31 (the name and extension donâ€™t matter), you can instruct **Tidy** to use it via
32 the command line option `-config config.txt`; for example:
33 
34 `tidy -config config.txt file1.html file2.html`
35 
36 Alternatively, you can name the default config file via the environment variable
37 named `HTML_TIDY`, the value of which is the absolute path for the config file.
38 
39 You can also set config options on the command line by preceding the name of the
40 option immediately (no intervening space) with the string "`--`"; for example:
41 
42 `tidy --break-before-br true --show-warnings false`
43 
44 You can find Quick Reference documentation for your version of **Tidy** that describe the
45 full set of configuration options on our [API and Quick Reference Page][1].
46 
47 
48 ## Sample Configuration File
49 
50 The following is an example of a **Tidy** config file.
51 
52 ~~~
53 // sample config file for HTML tidy
54 indent: auto
55 indent-spaces: 2
56 wrap: 72
57 markup: yes
58 output-xml: no
59 input-xml: no
60 show-warnings: yes
61 numeric-entities: yes
62 quote-marks: yes
63 quote-nbsp: yes
64 quote-ampersand: no
65 break-before-br: no
66 uppercase-tags: no
67 uppercase-attributes: no
68 char-encoding: latin1
69 new-inline-tags: cfif, cfelse, math, mroot,
70  mrow, mi, mn, mo, msqrt, mfrac, msubsup, munderover,
71  munder, mover, mmultiscripts, msup, msub, mtext,
72  mprescripts, mtable, mtr, mtd, mth
73 new-blocklevel-tags: cfoutput, cfquery
74 new-empty-tags: cfelse
75 ~~~
