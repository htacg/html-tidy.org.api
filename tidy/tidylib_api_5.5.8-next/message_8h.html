ctmbstr TY_() tidyErrorCodeAsKey ( uint  code)

LibTidy users may want to use TidyReportCallback to enable their own localization lookup features.

Because Tidy's report codes are enums the specific values can change over time. This function returns a string representing the enum value name that can be used as a lookup key independent of changing string values. TidyReportCallback will return this general string as the report message key.

uint TY_() tidyErrorCodeFromKey ( ctmbstr  code)

Given an error code string, return the integer value of it, or UINT_MAX as an error flag.
